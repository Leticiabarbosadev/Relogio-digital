⏰ Relógio Digital

O projeto Relógio Digital é uma aplicação web desenvolvida com HTML, CSS e JavaScript, que exibe a hora atual do sistema em tempo real, separando horas, minutos e segundos de forma organizada e dinâmica.

A estrutura do projeto segue boas práticas de organização, com separação de responsabilidades:
index.html → Estrutura da aplicação
assets/css/style.css → Estilização da interface
assets/js/script.js → Lógica de atualização do horário

A interface é composta por três blocos principais, cada um responsável por exibir:
Horas
Minutos
Segundos

A atualização é feita dinamicamente via JavaScript, manipulando o DOM através dos ids (horas, minutos, segundos), proporcionando atualização em tempo real.
